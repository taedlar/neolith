# make_path_absolute()
## NAME
**make_path_absolute** - resolve relative path name

## SYNOPSIS
~~~cxx
string make_path_absolute (string rel_path);
~~~

## DESCRIPTION
This master apply is called by the [ed()](../../efuns/ed.md) efun to resolve relative path names of a file to read/write, to an absolute path name.
