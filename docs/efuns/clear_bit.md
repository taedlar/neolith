# clear_bit()
## NAME
**clear_bit** - zero a bit in a bit string

## SYNOPSIS
~~~cxx
string clear_bit( string str, int n );
~~~

## DESCRIPTION
Return the new string where bit **n** is cleared in string
**str**. Note that the old string **str** is not modified.
