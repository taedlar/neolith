# query_shadowing()
## NAME
shadowp() - determine whether or not a given object it
shadowing another

## SYNOPSIS
~~~cxx
object shadowp( object ob );
~~~

## DESCRIPTION
Returns the object that `ob' is shadowing, or zero (0) if it
is not shadowing any object.
