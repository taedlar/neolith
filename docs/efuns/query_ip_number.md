# query_ip_number()
## NAME
**query_ip_number** - return the ip number for a player object

## SYNOPSIS
~~~cxx
string query_ip_number( object ob );
~~~

## DESCRIPTION
Return the ip-number (dotted decimal form) for player `ob'.
