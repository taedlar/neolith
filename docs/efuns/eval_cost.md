# eval_cost()
## NAME
**eval_cost** - returns the evaluation cost remaining

## SYNOPSIS
~~~cxx
void eval_cost()
~~~

## DESCRIPTION
eval_cost() returns the number of instructions that can be
executed before the driver decides it is in an infinite
loop.
