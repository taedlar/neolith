# reset_eval_cost()
## NAME
**reset_eval_cost** - resets the evaluation cost remaining

## SYNOPSIS
~~~cxx
void reset_eval_cost();
~~~

## DESCRIPTION
reset_eval_cost(), resets the evaluation cost remaining to
the maximum evaluation cost.

> [!NOTE]
> This is an alias of `set_eval_limit(0)`.
