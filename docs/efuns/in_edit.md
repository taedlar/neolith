# in_edit()
## NAME
**in_edit** - determine if a player is in the editor

## SYNOPSIS
~~~cxx
string in_edit( object default : F_THIS_OBJECT );
~~~

## DESCRIPTION
If the given object is in the editor, the file being edited
is returned, else zero.
