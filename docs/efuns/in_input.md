# in_input
## NAME
in_input() - determines if a player is inputting to an `input_to`.

## SYNOPSIS
~~~cxx
int in_input( object default : F_THIS_OBJECT );
~~~

## DESCRIPTION
Returns 1 if the object is currently inputting to an `input_to` or `get_char`.
