# interactive
## NAME
interactive() - detects whether or not a given object is an interactive

## SYNOPSIS
~~~cxx
int interactive( object ob );
~~~

## DESCRIPTION
Return non-zero if *ob* is an interactive player.
0 will be returned if he is link dead.
