# intp()
## NAME
**intp** - determine whether or not a given variable is an integer

## SYNOPSIS
~~~cxx
int intp (mixed arg);
~~~

## DESCRIPTION
Return `1` if **arg** is an integer number and zero (`0`) otherwise.
