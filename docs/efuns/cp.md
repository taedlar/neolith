# cp()
## NAME
**cp** - copy a file

## SYNOPSIS
~~~cxx
int cp( string src, string dst );
~~~

## DESCRIPTION
Copies the file **src** to the file **dst**.

## RETURN VALUE
Returns 1 for success, returns a negative value in case of
failure.
