# max_eval_cost()
## NAME
**max_eval_cost** - returns the maximum evaluation cost

## SYNOPSIS
~~~cxx
void max_eval_cost()
~~~

## DESCRIPTION
max_eval_cost() returns the number of instructions that can
be executed before the driver decides it is in an infinite
loop.
