# rename()
## NAME
**rename** - rename a file

## SYNOPSIS
~~~cxx
int rename( string src, string dst );
~~~

## DESCRIPTION
Renames the file **src** to **dst**.

## RETURN VALUE
rename() returns zero (0) to indicate success.  Nonzero
indicates failure.
